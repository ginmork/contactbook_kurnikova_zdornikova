# ContactBook 

Краткое и надежное Android-приложение на **Kotlin** и **Jetpack Compose**, демонстрирующее работу с **Intents** для связи с организацией.

## Функционал

Приложение отображает один экран с четырьмя кнопками:
1. **Позвонить** — открывает системный номеронабиратель с предзаполненным телефоном.
2. **Написать email** — запускает почтовый клиент, автоматически заполняя адресата (`mailto:`) и тему письма.
3. **Показать офис на карте** — открывает картографическое приложение (Google Maps, Яндекс.Карты) и устанавливает маркер с названием офиса по точным координатам.
4. **Поделиться контактом** — вызывает системное диалоговое окно Android Share Sheet для отправки текстовых контактов в любой мессенджер или заметки.
